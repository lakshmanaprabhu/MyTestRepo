Step 1: Go To Projects Directory 
	cd Data/D/Projects
Step 2:	Initailize the Git (Only once in a System)
	git init
Step 3:	Add the updated Projects Directory files to Git
	git add * 
	or
	git add filename1 filename2
Step 4:	Add the Comments to every Git Commit
	git commit -am "Comments"
	eg:
	git commit -am "first Commit"
Step 5:	Add the Local Git Changes to Remote Server (Only once for a Label, If you want other Label need this command)
		Create a Label for us in the server to identify the our git
	git remote add LABELNAME username@serverip:git_location
	eg:
	git remote add prabhu newgit@192.168.1.200:repo.git
Step 6:	Push the Local Label to Server (Only for creating the label in server) Next time Skip Step 6.
		Create a Label in Server for us
	git push LABELNAME HEAD:SERVER_LABEL
	eg:
	git push prabhu HEAD:Androidprojects
Step 7: push the content to Server (After the Step 6, Not needed on the First Time)
	git push LABELNAME master:refs/heads/SERVER_LABEL
	eg:
	git push prabhu master:refs/heads/Androidprojects

Step 8:	Check the Logs (If Needed)
	git log 	=>	Shows only commit ids
	git log -p  => 	Shows Every Differences with previous Commit
